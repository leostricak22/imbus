# Imbus

# Što je Imbus?

Imbus je mobilna aplikacija koja omogućuje korisnicima da pronađu majstore unutar Hrvatske, a razvijena je kao dio TVZ Mc2 natjecanja. Aplikacija je izrađena putem Spring Boot i React Native.
